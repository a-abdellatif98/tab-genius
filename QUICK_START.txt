TAB GENIUS - QUICK START GUIDE
==============================

INSTALLATION (5 MINUTES)
------------------------

Step 1: Add Icons (REQUIRED)
   - Go to the 'icons' folder
   - Read ICONS_NEEDED.txt for instructions
   - Add three icon files (icon16.png, icon48.png, icon128.png)
   - Simplest: Screenshot a brain emoji üß† and resize it!

Step 2: Install in Chrome
   - Open Chrome
   - Go to: chrome://extensions/
   - Enable "Developer mode" (top-right toggle)
   - Click "Load unpacked"
   - Select the entire 'tab-genius-extension' folder
   - Done! The extension is now installed

Step 3: Pin to Toolbar
   - Click the puzzle icon in Chrome toolbar
   - Find "Tab Genius"
   - Click the pin icon
   - Now you can access it anytime!

BASIC USAGE (MANUAL MODE)
--------------------------

1. Open many tabs in Chrome (or use your existing tabs)

2. Click the Tab Genius icon in toolbar

3. You'll see:
   - Total tabs
   - How many to close
   - How many to bookmark
   - Categories summary

4. Click "View Full Dashboard" button

5. Review each category:
   - Tabs are automatically categorized
   - Each tab has a suggested action (Keep/Close/Bookmark)
   - You can change the action using the dropdown

6. Select tabs you want to process (checkboxes)

7. Click "Execute Selected" button

8. Confirm the action

9. A report file will be downloaded automatically!

USING CLAUDE API (AUTO MODE)
-----------------------------

1. Get API Key:
   - Go to: https://console.anthropic.com/
   - Sign up or log in
   - Create an API key
   - Copy the key (starts with sk-ant-...)

2. Configure Extension:
   - Click Tab Genius icon
   - Click Settings (gear icon)
   - Paste your API key
   - Click "Test API Key"
   - Click "Save Settings"

3. Use Auto Mode:
   - Open the dashboard
   - Click "Auto Mode (Claude)" button
   - Wait for Claude to analyze
   - Review the AI suggestions
   - Execute when ready!
